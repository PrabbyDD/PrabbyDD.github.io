Simple Ray Tracer

The second ray tracer I have built! Still a work in progress, but the basic ray tracing is functional and is equipped with many types of materials and shapes. The thumbnail image was rendered with this raytracer. Many optimizations to come! Link to the github repo!

My First Compiler! (in C++)

The purpose of this project was to get better at C++ and gain a better appreciation of programming languages at a low level. This project was heavily guided by the Pixeled series on youtube and Cals CS164 course notes. Play around with the language hydrogen! Link to the github repo!

Conflict Based Search (in C++)

This was a smaller project meant to understand how the multi agent path planning algorithm Conflict based search worked. This algorithm uses A* as a subroutine and is a fundamental algorithm in MAPP for robotics and other applications. Link to the github repo!

Mandelbrot and parallelism (in C++)

I wanted a simple project to better understand the relationship between threading parallelism and SIMD extensions. I got a 66x speedup using threading with SIMD vs not using them, which is nice! You can zoom in on the image and move around with the arrow keys if you run the executable, but the quality on the image is not that good, but you can scroll for quite a while since the maximum iterations is initially at 1000. Its pretty cool even if its imperfect, and I learned a lot! Link to the github repo!
